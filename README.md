# 停车场信号量教学程序

使用 Python 多线程和信号量（Semaphore）模拟停车场管理的教学示例程序。

## 功能特点

- **三个独立执行单元**：入口线程、出口线程、管理员线程分别模拟不同角色
- **信号量保护车位**：`threading.Semaphore` 保证车位不会被抢成负数
- **满场等待**：车位已满时车辆在入口排队等待
- **车辆离场释放车位**：车辆离开后释放信号量，等待队列中的车辆可进场
- **管理员关闭入口**：管理员可随时关闭入口，阻止新车进入
- **在场车辆追踪**：通过集合记录已进场车辆，未进场车辆无法被误记为离场

## 并发原理解析

| 原语 | 用途 |
|------|------|
| `threading.Semaphore(capacity)` | 控制可用车位数量，`acquire()` 获取，`release()` 释放 |
| `threading.Lock` | 保护 `occupied` 计数器和 `parked_cars` 集合的互斥访问 |
| `threading.Event` | 管理员通过 `clear()` 关闭入口，入口线程通过 `is_set()` 检查状态 |

## 快速开始

### 本地运行

需要 Python 3.7+。

```bash
python3 parking_lot.py
```

### 使用 Docker Compose 运行

```bash
docker compose up --build
```

运行完毕后清理：

```bash
docker compose down
```

### 仅使用 Docker 运行

```bash
docker build -t parking-lot .
docker run --rm parking-lot
```
